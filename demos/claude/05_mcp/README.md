> **상태:** 자리만 잡아둔 상태. 실제 데모 콘텐츠는 다음 단계에서 채웁니다.

## 이 데모가 보여줄 것

- MCP(Model Context Protocol)는 Claude를 **외부 서비스**(Jira, Slack, Drive 등)와 연결하는 표준 프로토콜
- 출시는 가장 빨랐지만(2024-11), 시연 흐름에서는 마지막에 두는 이유: 외부 의존성이 있어 환경 셋업이 필요하기 때문
- "AI가 코드를 짜는 것"을 넘어 "AI가 우리 회사 시스템 안에서 일하는 것"을 가능하게 한다는 것
